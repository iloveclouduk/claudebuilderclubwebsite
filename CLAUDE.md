# Claude Code Instructions

## Development Server
- **NEVER run `npm run dev`** - The development server is already running
- The project is always available at the running dev server
- Do not attempt to start additional dev servers

## Project Details
- Next.js 16.1.6 with Turbopack
- Tailwind CSS 4
- TypeScript
- shadcn/ui components

## Commands to Avoid
- `npm run dev` (already running)
- `next dev` (already running)
- Any commands that would start additional servers

## Safe Commands
- `npm run build` (for production builds)
- `npm run lint` (for linting)
- `npm install <package>` (for adding dependencies)
